      NuaCode Cheatsheet — Shortcuts
      - Save: Ctrl/Cmd + S
      - Save All: Ctrl/Cmd + Shift + S
      - Run Preview: Ctrl/Cmd + R or Ctrl/Cmd + B
      - Stop Preview: Ctrl/Cmd + Shift + R
      - Duplicate Line/Selection: Ctrl/Cmd + D
      - Toggle Comment: Ctrl/Cmd + /
      - Undo: Ctrl/Cmd + Z
      - Redo: Ctrl/Cmd + Shift + Z
      - Find/Replace: Ctrl/Cmd + F
      - Toggle Outline: Ctrl/Cmd + K O
      Selection-aware menu items appear only when you have text selected.
